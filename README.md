# Biometric Update Data Analysis

This project is based on Aadhaar biometric update data.
The analysis is done using Python and visualized using Power BI.

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Power BI

## What is in this project
- State-wise biometric update analysis
- District-wise biometric update analysis
- Age-wise biometric update analysis
- Date-wise trend analysis

## Purpose
This project is created for learning data analysis and visualization.
